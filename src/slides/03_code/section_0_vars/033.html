Abstracting Our Mixin

  1. Create a new branch in your GitHub repo called "fpo-mixin"

  2. Create a new fo file called 'src/fpo.scss'

  3. Add arguments to the mixin for any CSS properties that you think a user would want to change.

  4. Commit your changes, and sync them to GitHub.

@mixin fpo() {
  background-color: slategray;
  position: relative;

  &:before {
    content: '';
    display: block;
    padding-bottom: 100%;
  }
  
  &:after {
    content: 'FPO';
    color: #fff;
    font: 20px/1 Helvetica;
    position: absolute;
    top: 50%;
    left: 50%;
    transform: translate(-50%, -50%);
  }
}
