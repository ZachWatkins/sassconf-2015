Mixin Arguments

Mixin arguments allow you to customize the behavior of a mixin.

@mixin button($bg-color) {
  background-color: $bg-color;
}

Multiple arguments are separated by commas, and you can set default values the same way you define variables:

@mixin button($bg-color, $text-color:#FFF) {
  background-color: $bg-color;
  color: $text-color;
}
