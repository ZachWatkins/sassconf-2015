Mixin Arguments, Continued

For truly optional arguments, use null as the default value.

CSS properites with null as a value won't be output by Sass.

@mixin button($bg-color, $text-color:#FFF, $border: null) {
  border: $border;
  background-color: $bg-color;
  color: $text-color;
}

.button { @include button(#C0FFEE); }

Compiles to:

.button {
  background-color: #C0FFEE;
  color: #FFF;
}

Required arguments have to be ordered before optional ones.
